## 2. 💻 Tecnologias usadas
* Vue;
* Axios;
* Bulma
* [PokéAPI](https://pokeapi.co/)


## 3. 🎉 Para rodar o projeto
1. Instale todas es tec's usadas junto do Node JS:

    ```
    npm install
    ```
2. Rode o aplicativo

    ```
    npm run serve
    ```
3. Entre em https://localhost:8080 e teste o aplicativo
